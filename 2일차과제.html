미레도레가 만든 코뮤니티

코뮤니티는 코딩의 신 코뮤가 만든 코딩 공부 커뮤니티이다. 코딩 독학 콘텐츠가 있으며, 모여서 각자 코딩하는 모각코가 매우 인기가 많다.


코뮤니티 모각코

웹 개발 기초 (html, css)

  • 웹 개발을 배우고 싶은 사람들을 위한 매우 기초적인 과정
  • 웹의 기초부터 html과 css 등의 기초적인 태그까지의 커리큘럼

    • 웹 개발 심화 (html, css)

  • 웹 개발의 기초에서 좀 더 심화된 내용을 배우고 싶은 사람들을 위한 과정
  • 웹의 기초를 짧게 다진 후에 html과 css 등에서 쓰이는 유용한 태그까지의 커리큘럼

    • 웹 개발 심화 (자바스크립트 편)

  • html과 css로 만들어진 웹에 동적 기능을 부여하는 방법을 배우고 싶은 사람들을 위한 과정
  • 자바스크립트의 기본 개념부터 유용한 기능까지의 커리큘럼
